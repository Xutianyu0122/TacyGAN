# TacyGAN
Official implementation of:

**TacyGAN: Geometry-Consistent Sim-to-Real Transfer for Visuo-Tactile Sensing**

## Overview

TacyGAN is a geometry-consistent sim-to-real transfer framework for visuo-tactile sensing. The method explicitly preserves tactile contact geometry during image translation through:

* Depth Geometric Consistency Loss
* Contact State Consistency Loss
* Hybrid Multi-Attention U-Net (HMA-UNet)
* Multi-Scale Cross-Attentive Refinement Discriminator (MS-CARD)

The framework reduces the sim-to-real gap while maintaining the physical consistency required for downstream tactile perception tasks such as depth reconstruction and force estimation.
